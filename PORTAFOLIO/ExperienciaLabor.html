Proyecto PHP:Aplicación web para la gestión de Administradores, Docentes, Alumnos y materias perfecta para una institución educativa. Proyecto desarrollado en PHP y My SQL

Planner Travel: Aplicación Móvil su funcion es dar una guía a los turistas para encontrar lugares representativos que potencialmente podrían visitar como restaurantes, Hoteles, Actividades para realizar, sitios más visitados en este caso para Colombia.Proyecto desarrollado en Android Studio, Java y Autenticación con Firebase.
